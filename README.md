# Analyse des ventes – Mini Projet Data & BI

## Objectif
Analyser les ventes d’un magasin afin d’identifier les produits et périodes les plus performants.

## Données
Dataset Superstore Sales (Kaggle)

## Méthodologie
- Chargement des données
- Analyse descriptive
- Visualisation des ventes

## Résultats
Identification des catégories de produits les plus performantes.
